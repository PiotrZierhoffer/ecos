fs deldir

Name

fs deldir -- Delete filesystem directory

Synopsis

fs deldir {directory}

Arguments

NameTypeDescriptionDefault
directoryStringPathname to directory to delete. 

Description

This command deletes a directory from the filesystem. If the directory contains files or other directories then this command will fail.

Examples

Delete directory: 

RedBoot> fs list                                                                
212416 d---------  3 size    128 .                                              
212416 d---------  3 size    128 ..                                             
211392 d---------  2 size     96 tests                                          
210368 ----------  1 size   4096 image                                          
RedBoot> fs deldir tests                                                        
RedBoot> fs list                                                                
212416 d---------  2 size    128 .                                              
212416 d---------  2 size    128 ..                                             
210368 ----------  1 size   4096 image                                          
RedBoot>
